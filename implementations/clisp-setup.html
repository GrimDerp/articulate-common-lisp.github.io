CLISP Setup

The CLISP Common Lisp implementation is a good general-purpose interpreter licensed under the terms of the GNU GPLv2. Though not as popular as SBCL, it sports an intuitive user interface, a native compiler and an easy-to-use debugger.

It also includes various useful extensions to the official ANSI standard, such as foreign function calls and a socket interface. (Note however that using implementation-specific features binds one to the GPL.)

Installing

Download one of the packages or pre-built binaries.

Installing on Ubuntu

On Ubuntu-based systems, you can install CLISP from the repositories:

sudo apt-get install clisp
